== Description ==

LionScripts' Webmaster Tools creates Wordpress Webmasters life really easy by linking various Online Webmaster Tools with the WordPress Admin Panel and allowing them to verify Various Webmasters Tools like Google, Bing, Yandex, Alexa, Pinterest etc., and put Google Analytics and Custom Site Analytics.

Webmaster Tools is World's first Wordpress plugin to provide all-in-one website verification interface, where you just need to enter the Meta tags to verify the website. Everything is really so simple, to make you feel installing "Webmaster Tools" as the first plugin on every WordPress website.

By verifying the Webmaster Tools at Google, Bing, Yandex, Alexa and Pinterest you will reach almost 99% of the audience and get maximum revenue from your website.

== Installation ==

1. Upload 'webmaster-tools' directory to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to **LionScripts** menu and then **Webmaster Tools** to configure
